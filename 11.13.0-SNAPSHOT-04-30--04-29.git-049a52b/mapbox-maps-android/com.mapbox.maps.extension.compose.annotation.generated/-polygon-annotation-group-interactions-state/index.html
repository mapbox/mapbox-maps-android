var isDraggable: Boolean

Set whether PolygonAnnotation holding this PolygonAnnotationGroupInteractionsState should be draggable, meaning it can be dragged across the screen when touched and moved. Note: If this param is used when PolygonAnnotation is also part of the cluster, then once PolygonAnnotation is dragged, it moves out of the cluster and can't be added back to it and is rendered as a separate PolygonAnnotation.
