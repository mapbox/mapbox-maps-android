defaultPluginList

val defaultPluginList: List<Plugin>

Default map view plugin registry. All Mapbox plugins are present in this list.

If specific plugins are required to be added to MapView on startup consider creating MapView programmatically specifying plugins.

If creating MapView from xml - all plugins listed here will be applied to given MapView on startup.
