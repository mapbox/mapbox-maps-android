PAUSE_IN_BACKGROUND

val PAUSE_IN_BACKGROUND: Int = 2

Supported for Android only. If set and application also has "com.mapbox.common.http.pause_requests_on_demand" setting in manifest the request will be performed in the foreground only. In case application is in the background request will be queued.
