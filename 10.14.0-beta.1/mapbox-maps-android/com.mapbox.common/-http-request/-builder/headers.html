headers

open fun headers(@NonNull headers: HashMap<String, String>): HttpRequest.Builder
Content copied to clipboard

HTTP headers to include. If you provide `User-Agent` header using this `headers` field, then resulting `User-Agent` will also contain information about application, platform, device, and Common SDK's version. If you provide duplicate headers then `request` call will be immediately failed.
