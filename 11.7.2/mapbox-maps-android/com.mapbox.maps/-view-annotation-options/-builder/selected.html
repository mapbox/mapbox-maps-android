open fun selected(@Nullable selected: Boolean): ViewAnnotationOptions.Builder

Specifies if this view annotation is selected meaning it should be placed on top of others. When adding new annotations, if `selected` is not explicitly set, default value `false` will be applied. When updating existing annotations, if `selected` is not explicitly set, the current value will be retained.
